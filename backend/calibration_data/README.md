# Calibration Data

This directory stores empirically derived calibration data from benchmark evaluations.

## lfw_calibration.json

Generated by `scripts/calibrate_lfw.py`. Contains:

- **FAR/FRR** at every evaluated cosine threshold (from the LFW benchmark)
- **EER** (Equal Error Rate) for each scoring tier
- **Optimal fusion weights** derived via grid search over the LFW pair dataset
- **Benchmark provenance**: source URL, pair counts, timestamp

### Chain of Custody

The `pairs.txt` file is downloaded at runtime directly from:
```
http://vis-www.cs.umass.edu/lfw/pairs.txt
```

No local modifications are applied. The calibration script processes the identical pipeline (CLAHE → align → ArcFace/geometric/LBP) used in production to ensure the calibration data accurately represents the system's operating characteristics.

### Regenerating

```bash
cd backend
python ../scripts/calibrate_lfw.py
```

Or via Cloud Run Job:
```bash
gcloud run jobs execute lfw-calibration-job --region us-central1
```
